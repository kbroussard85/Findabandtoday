# Roadmap: FABT (Automated AI Talent Agency)

This roadmap outlines the development of the FABT platform, from initial technical foundation to advanced AI-driven negotiation agents.

## Phases

- [x] **Phase 1: Identity & Geo-Foundation** - Core stack, Auth0 zero-knowledge, and PostGIS setup.
- [x] **Phase 2: Directory & Premium UX** - Onboarding, discovery grid, and freemium gated access.
- [x] **Phase 3: Transaction Engine** - Stripe Connect, automated contracts, and payout automation.
- [x] **Phase 3.5: Venue Marketplace & Dashboard Pivot** - Tinder-style booking, Agreement Vault, and Venue-specific UX.
- [x] **Phase 3.8: Escrow & Stripe Protection** - Manual capture, platform fee escrow, and deposit management.
- [x] **Phase 4: Agentic Negotiation** - LangGraph agents for auto-negotiation and tour optimization.

---

## Phase Details

### Phase 3.8: Escrow & Stripe Protection
**Goal**: Secure funds and protect both parties using Stripe's manual capture and escrow logic.
**Success Criteria**:
  1. $50 flat escrow or 5% fee is authorized upon booking submission.
  2. Funds are held in platform escrow until show completion or manual release.
  3. No-show logic handles automated payout/refund based on contract terms.
