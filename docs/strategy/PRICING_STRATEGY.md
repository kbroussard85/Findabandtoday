# FABT Pricing & Tier Strategy

## Overview
Findabandtoday (FABT) uses a "SaaS + Transaction Fee" model to balance predictable recurring revenue with the scalable upside of a high-volume marketplace.

## 1. Artist Tiers (The Gig Seekers)
| Tier | Price | Features |
| :--- | :--- | :--- |
| **Free** | $0 | Basic EPK, Directory listing, 2 apps/month. |
| **Standard** | $9.99/mo | Unlimited apps, Contract automation, Tax (I-9) tools. |
| **Pro** | $24.99/mo | Priority matching, 30-day "Gig Maximizer" automation. |

## 2. Venue Tiers (The Demand)
| Tier | Price | Features |
| :--- | :--- | :--- |
| **Starter** | $29.99/mo | 1 Stage management, Automated payouts (+1% fee). |
| **Growth** | $79.99/mo | Multi-stage, Custom contracts, Team access (+0.5% fee). |

## 3. Specialized Services (High-Margin Add-ons)
*   **Media Evaluation ($49.99):** One-time review of social media and EPK quality.
*   **Tour Booking ($229.00):** Manual coordination for 10+ city routing.
*   **30-Day Maximizer ($29.99):** AI-powered gig matching boost for 1 month.

## Strategy Rationale
*   **Monetize Utility, Give Away Presence:** Keep EPKs free to build the largest directory in the industry. Charge when the user attempts to extract value (apply for a gig).
*   **The Compliance Shield:** Position the subscription as a legal/tax convenience tool rather than just a "booking" tool to increase retention.
